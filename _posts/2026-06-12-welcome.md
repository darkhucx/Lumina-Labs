**Lumina Labs** 自动采集 AI 前沿博客文章，经机器翻译后由人工质检，质检通过的篇目发布到这里。

所有内容均来自公开来源，本站仅提供中文摘译与索引，版权归原作者所有。每篇文章顶部均标注 **原文链接**，建议对照原文阅读。

> 这是站点的首篇示例文章，用于验证列表页与文章页两套模板的渲染（标题 / 来源 / 采集时间 / 原文链接 / 标签 / 中文正文）。

## 工作流

1. dk-kanban 定时采集 → 机器翻译 → 写入 Obsidian 收件夹（`status: draft`）。
2. 人工在 Obsidian 里质检，通过的改 `status: published`。
3. 跑 `./publish.sh` 把 published 篇拷进本仓 `_posts/` → `git push`。
4. GitHub Action 自动 `jekyll build` 并部署到 GitHub Pages。
